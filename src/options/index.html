欢迎使用 Lens Translator

默认使用 Chrome 内置翻译（零配置）。也可随时改用外部 LLM。

  1. 选择语言 设置源语言与目标语言，并确认 Chrome 语言包就绪。
  2. 可选：外部 LLM 需要更高质量或图片翻译时，填写兼容 OpenAI 的接口。
  3. 开始翻译 点击扩展图标开关透镜，或使用快捷键 / 划词翻译。

PREFERENCES

设置

配置任意语言对、引擎与页面呈现。

基础翻译

控制翻译透镜的语言、引擎和交互方式。默认使用 Chrome 内置翻译。

翻译语言

任意源语言 → 目标语言。列表显示名称与 BCP 47 代码；Chrome 内置模式需语言包就绪。

→

透镜文本引擎

两种引擎互不兜底。默认 Chrome 内置；失败时会提示下载语言包或改用外部 LLM。

划词翻译

开启后，选中页面文字即显示译文浮层；取消选择立即消失。不影响透镜与整页模式。

自动预译可见区

滚动或页面更新时预先翻译透镜可能指向的文本。外部模式会主动发送文本。

页面悬浮气泡

网页右侧的快捷控制条。关闭后仍可通过扩展图标、快捷键与设置页操作。

透镜宽度

调整悬浮翻译面板的固定宽度。

透镜快捷键

短按保持打开，按住时临时查看；Mac 上 Option 对应 Alt。

Option+Shift+L 请按下组合键，Esc 取消

整页双语

保留原文，在对应文本块下追加译文，并持续处理动态内容。

整页翻译引擎

Chrome 内置速度更快；外部 LLM 适合需要更强上下文理解的页面。

服务商

根据 Base URL 和模型自动识别兼容参数。

接口地址

远程地址必须使用 HTTPS；本地模型可使用 localhost HTTP。

API Key

密钥保存在 chrome.storage.local；留空保存会保留已有密钥。

推理强度

翻译通常不需要深度推理，关闭或降低可减少等待。

连通性测试

文本：发送一次最小翻译请求。图片：探测模型是否支持 image_url 多模态输入。

网站管理

暂停的网站不会启动透镜、划词、自动预译或整页翻译。

暂停名单

输入主机名并用逗号分隔；也可在快捷控制面板中暂停当前网站。
